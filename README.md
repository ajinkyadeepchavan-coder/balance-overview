# balance-overview
CUSTOMER INSIGHTS AND BALANCE OVERVIEW

OVERVIEW
This project is an interactive Power BI dashboard developed to analyze customer demographics and banking performance. The dashboard provides insights into customer distribution, account balances, tenure, age, gender, job classification, and regional performance to support business decision-making.

OBJECTIVES-
Calculate Total Bank Balance.
Calculate Total Customers.
Analyze Gender Distribution of Customers.
Determine Average Customer Age.
Analyze Bank Balance by Region.
Analyze Bank Balance by Job Classification.
Calculate Average Customer Tenure.
Calculate Total Customers by Region.
Compare Bank Balance by Gender.
Create an interactive dashboard using all visualizations.

TOOLS USED-
Power BI Desktop
Microsoft Excel

KEY INSIGHTS-
Identified regions with the highest customer balances.
Compared customer distribution across different regions.
Analyzed gender-wise balance distribution.
Evaluated customer tenure and average age.
Identified job categories contributing the highest balances.
